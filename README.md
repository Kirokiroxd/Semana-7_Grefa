 Tienda Marcos

Sistema web de tienda online desarrollado con PHP + MySQL (XAMPP) que permite:

Registro e inicio de sesión de usuarios

Visualización de productos

Carrito de compras

Generación de órdenes

Visualización de facturas

 Descarga de factura en PDF usando JavaScript (jsPDF)

 Tecnologías utilizadas

PHP 8+

MySQL

XAMPP (Apache + MySQL)

HTML5

CSS3

JavaScript

jsPDF (para generar PDF desde el navegador)
